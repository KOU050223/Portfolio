# KOUのPortfolioサイト

React + ViteからNext.jsへの移行プロジェクト

## 技術スタック

- **フレームワーク**: Next.js 15.4.4 (App Router)
- **UI**: shadcn/ui + Tailwind CSS
- **言語**: TypeScript
- **移行元**: React + Vite + Chakra UI

## 開発環境

```bash
# 開発サーバー起動
npm run dev

# ビルド
npm run build

# 本番サーバー起動
npm start

# ESLintチェック
npm run lint
```

開発サーバー: [http://localhost:3000](http://localhost:3000)
